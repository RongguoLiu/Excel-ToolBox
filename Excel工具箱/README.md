# Excel-Toolbox
一个正在成长的Excel VSTO工具集，用于提高生活幸福度。

以C#编写，需要.Net Framework 4.6， 经测试在Excel 2013/ 2016/ 365 x86中能正常运行。欢迎在其他平台进行测试。

## 安装说明

下载 `.zip` 包后，请

* 要么用第三方软件解压

* 要么 `.zip-属性-勾选“解除锁定”-确定` ，再用Windows Explorer解压

然后运行 `setup.exe` ，否则，很可能遭遇错误……
